**Возможности:**

**1. Лента сделок в реальном времени:**
- WebSocket-подобная симуляция (обновление каждые 5 сек)
- Авто-обновление с контролем включения/выключения
- Живые метрики и статистика

**2. Типы сделок:**
- 💰 **Sale** - прямые продажи
- 📈 **Auction** - аукционные торги
- 🏷️ **Offer** - предложения
- ⚡ **Bid** - ставки

**3. Детальная информация:**
- Произведение (название, художник, категория)
- Цена с историей изменений
- Продавец и покупатель
- Статус сделки (активна/завершена/в ожидании)
- Тренд (рост/падение/стабильно)
- Просмотры и избранное
- Локация сделки

**4. Фильтрация и поиск:**
- По типу сделки
- По статусу
- По ценовому диапазону
- По категории
- Текстовый поиск
- 5 вкладок для быстрой навигации

**5. Статистика:**
- Всего сделок в ленте
- Количество активных
- Общий объём сделок
- Средний чек

**6. Интерактивность:**
- Клик на карточку → Цифровой паспорт
- Кнопка "Поделиться"
- Реакции (просмотры, избранное)
- Анимации появления новых сделок

**Технологии:**
- React + TypeScript
- Framer Motion для анимаций
- AnimatePresence для плавного добавления/удаления
- Simulated WebSocket (setInterval)

**Метрики:**
- 21,655 символов
- 50+ сделок в ленте
- Real-time обновления
- Автоматическая анимация

### 3.3 Notification System ✅
**Файл:** `client/src/pages/NotificationCenter.tsx` (21.8 KB)  
**Маршрут:** `/notifications`

**Возможности:**

**1. Центр уведомлений:**
- Единый центр управления
- Real-time обновления (каждые 15 сек)
- До 100 уведомлений в истории
- Статистика: всего/непрочитанных/срочных/за сегодня

**2. Девять типов уведомлений:**

📊 **Deals (Сделки):**
- Новые продажи
- Завершённые сделки
- Быстрые продажи

📈 **Auctions (Аукционы):**
- Начало аукциона
- Новые ставки
- Результаты торгов

💰 **Price Alerts (Ценовые алерты):**
- Снижение цен
- Достижение целевой цены
- Рост стоимости

💬 **Messages (Сообщения):**
- Новые сообщения
- Ответы консультантов
- Непрочитанные

📅 **Events (События):**
- Напоминания о выставках
- Новые мероприятия
- Подтверждение билетов

⚙️ **System (Системные):**
- Обновления
- Статистика просмотров
- Резервное копирование

🤖 **Recommendations (AI-рекомендации):**
- Новые работы для коллекции
- Инвестиционные возможности
- Похожие произведения

❤️ **Favorites (Избранное):**
- Новые работы от избранных художников
- Доступность из wishlist
- Участие в выставках

🏆 **Achievements (Достижения):**
- Разблокированные достижения
- Новые уровни
- Награды и баллы

**3. Приоритеты:**
- **Low** - обычные (без метки)
- **Medium** - средние (синий badge)
- **High** - важные (оранжевый badge)
- **Urgent** - срочные (красный badge)

**4. Управление:**
- Отметить как прочитанное
- Отметить все как прочитанные
- Удалить уведомление
- Очистить все
- Поиск по тексту
- Фильтрация по типу (10 вкладок)

**5. Настройки уведомлений:**
- Включение/выключение по типам
- Каналы доставки:
  - 📧 Email уведомления
  - 📱 Push уведомления
  - 📞 SMS уведомления
- Индивидуальный контроль для каждого типа

**6. Интерактивность:**
- Кнопки действий (Открыть, Прочитано, Удалить)
- Переход к связанным страницам
- Анимации появления/удаления
- Визуальные индикаторы непрочитанных

**Технологии:**
- React + TypeScript
- Framer Motion
- Switch components (shadcn/ui)
- Real-time simulation

**Метрики:**
- 21,817 символов
- 9 типов уведомлений
- 10 вкладок фильтрации
- Полная настройка
