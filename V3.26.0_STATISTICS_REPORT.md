### 3. 📈 Группировки данных

#### **По жанрам:**
```
interface GenreData {
  name: string;        // Название жанра
  volume: number;      // Объём продаж
  growth: number;      // Рост за период (%)
  count: number;       // Количество произведений
}
```

#### **По художникам:**
```
interface ArtistData {
  name: string;        // ФИО художника
  sales: number;       // Количество продаж
  revenue: number;     // Общая выручка
  growth: number;      // Рост выручки (%)
  avatar?: string;     // URL аватара
}
```

#### **По галереям:**
```
interface GalleryData {
  name: string;        // Название галереи
  artworks: number;    // Количество произведений
  revenue: number;     // Выручка
  rating: number;      // Рейтинг 0-5
}
```

#### **По событиям:**
```
interface EventData {
  name: string;        // Название события
  date: string;        // Дата проведения
  participants: number;// Участников
  volume: number;      // Объём торгов
}
```

#### **По ценовым диапазонам:**
```
interface PriceRangeData {
  range: string;       // Диапазон (< $1K, $1K-5K, и т.д.)
  count: number;       // Количество работ
  avgPrice: number;    // Средняя цена
}
```

5 категорий:
- < $1,000 (456 работ, сред. $650)
- $1,000 - $5,000 (289 работ, сред. $2,800)
- $5,000 - $10,000 (145 работ, сред. $7,200)
- $10,000 - $50,000 (87 работ, сред. $25,000)
- > $50,000 (34 работы, сред. $85,000)

#### **По стилям:**
```
interface StyleData {
  style: string;       // Название стиля
  demand: number;      // Доля рынка (%)
  trend: 'up' | 'down' | 'stable'; // Тренд
}
```

С индикаторами тренда:
- ↑ (up) - растущий тренд
- ↓ (down) - падающий тренд
- → (stable) - стабильный
