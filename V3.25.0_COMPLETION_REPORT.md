## ✅ Задача 2: Website Builder для галерей, партнёров и кураторов

### Что создано:
Полнофункциональный Website Builder с drag-and-drop интерфейсом для создания персональных сайтов галерей.

### Основные возможности:

#### 1. **Библиотека компонентов (6 типов)**:
- 📐 **Hero секция** - большой заголовок с фоновым изображением и CTA кнопкой
- 📝 **Текстовый блок** - параграфы и заголовки (h1, h2, h3, p)
- 🖼️ **Изображение** - одиночное изображение с подписью
- 🎨 **Галерея** - сетка изображений (настраиваемое количество колонок)
- 👤 **О нас** - секция с текстом и изображением
- 📞 **Контакты** - контактная информация (email, телефон, адрес)

#### 2. **Инструменты редактирования**:
- ⬆️⬇️ Перемещение компонентов вверх/вниз
- 📋 Копирование компонентов
- 🗑️ Удаление компонентов
- 👁️ Выделение выбранного компонента

#### 3. **Режимы предпросмотра**:
- 🖥️ **Desktop** - полная ширина
- 📱 **Tablet** - 768px
- 📱 **Mobile** - 375px
- Плавная анимация перехода между режимами

#### 4. **Управление страницами**:
- ➕ Создание нескольких страниц
- 📑 Вкладки для переключения между страницами
- 🏠 Главная страница по умолчанию

#### 5. **Настройки сайта**:
- 📛 Название сайта
- 🎨 Тема (primary/secondary цвета, шрифт, фон)
- 💾 Сохранение в localStorage
- 🌐 Публикация (в production - на реальный домен)

### Технические детали:
- **Файл**: `client/src/pages/WebsiteBuilder.tsx` (новый, 20KB)
- **Роут**: `/website-builder`
- **Компоненты**: React + TypeScript
- **Анимации**: Framer Motion
- **UI**: Shadcn/ui компоненты
- **Иконки**: Lucide React

### Архитектура данных:

- 📰 Иконка Newspaper
- Заголовок "Новости мира искусства"

Каждая карточка содержит:

- 📅 **Дата** - дата публикации (в формате "5 февраля")
- 📝 **Заголовок** - заголовок новости (до 2 строк)
- 📄 **Превью** - краткое описание (до 3 строк)
- 🔗 **Ссылка** - "Читать далее" с анимированной стрелкой

- 🟢 **Рынок** (market) - зелёный
