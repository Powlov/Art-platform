## Особенности

- Все аккаунты созданы с уникальными username
- Пароли захешированы с помощью bcrypt
- Каждая роль имеет свой собственный dashboard
- Artist role имеет связанный профиль artist в базе данных

## База Данных

- **Тип**: SQLite
- **Путь**: `/home/user/webapp/artbank.db`
- **Произведений**: 8 шт.
- **Новостей**: 3 статьи
- **Художников**: 1 профиль

## Как использовать

2. Выберите роль (Гость, Коллекционер, Художник и т.д.)
3. Заполните форму регистрации
4. После регистрации войдите с новыми учетными данными
